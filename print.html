swc: speedy web compiler

Introduction

What is swc?

swc is a super-fast alternative for babel and google closure compiler.

Installation

You can install swc via

npm i -D swc

or

yarn add --dev swc

This will download prebuilt binary instead of building swc if possible. Currently prebuilt binaries are provided only for mac (x64) / linux (x86_64) and for some of node js versions (6, 8, 10, 11).

To use swc in other environment, you should install nightly version of rust first. It can be done with

curl https://sh.rustup.rs -sSf | sh -s -- --default-toolchain=nightly

If you are using windows, please go to rustup.rs.

After installing rust, you can install swc via npm / yarn.

Usage

node-swc

Note: If you are not developing build tools, you can skip this.

const swc = require('swc');

const output = swc.transform('source code', {
    
});
output.code; // transformed code
output.map; // source map (in string)

const output2 = swc.transformFileSync(pathToFile, {
    
})
output2.code; // transformed code
output2.map; // source map (in string)
